# Coffee Sales Analysis in SQL

Tools: WINDOW FUNCTIONS, GROUP BY, ORDER BY, AGGREGATE FUNCTIONS

Using basic SQL functions I analyze basic trends in coffe sales dataset:
1. Total revenue overall
2. Most profitable type of coffee and total revenue for each coffee type
3. Most commonly bought type of coffee
4. Total revenue in each month
5. Daily revenue and 7-day rolling average 
6. Cash vs card sales

Dataset: [Coffee Sales](https://www.kaggle.com/datasets/ihelon/coffee-sales)
